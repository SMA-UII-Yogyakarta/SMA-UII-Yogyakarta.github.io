# Filosofi — SMAUII Developer Foundation

## Visi

**Membangun ekosistem developer dan open source contributor sejak SMA.**

SMAUII Developer Foundation bukan sekadar ekstrakurikuler atau klub hobi. Ini adalah **wadah serius** bagi siswa yang benar-benar berkomitmen menekuni teknologi — dari robotika/IoT, AI, data science, jaringan komputer, keamanan siber, hingga software engineering.

Kami percaya bahwa **passion dan komitmen sejak dini** adalah kunci untuk menghasilkan developer dan researcher berkelas dunia. Foundation ini dirancang untuk mengantarkan siswa ke:

- 🏆 **Olimpiade Nasional/Internasional** — OPSI, OSN, IOI, ICPC
- 🎓 **Perguruan Tinggi Top** — dengan portofolio dan pengalaman nyata
- 🌍 **Komunitas Open Source Global** — kontribusi ke proyek dunia nyata
- 💼 **Karir Teknologi** — dengan skill yang relevan dengan industri

---

## Prinsip Inti

### 1. **Open Source First**

Semua proyek foundation adalah open source. Kami percaya:
- Transparansi menghasilkan kualitas
- Kolaborasi lebih kuat dari kompetisi internal
- Kontribusi ke komunitas global adalah pembelajaran terbaik
- Portofolio GitHub adalah CV masa depan

### 2. **Learning by Building**

Tidak ada teori tanpa praktik. Setiap track memiliki:
- **Proyek nyata** — bukan sekadar tugas sekolah
- **Kontribusi open source** — ke proyek yang digunakan orang lain
- **Kompetisi** — OPSI, CTF, Kaggle, hackathon
- **Mentorship** — dari alumni dan praktisi industri

### 3. **Komunitas, Bukan Kelas**

Ini bukan kursus dengan kurikulum kaku. Ini adalah **komunitas** di mana:
- Senior membimbing junior
- Peer learning lebih efektif dari ceramah
- Setiap orang berkontribusi sesuai kemampuan
- Kolaborasi lintas track didorong

### 4. **Komitmen > Bakat**

Kami tidak mencari "anak jenius". Kami mencari siswa yang:
- **Konsisten** — hadir dan berkontribusi rutin
- **Curious** — selalu ingin belajar lebih dalam
- **Collaborative** — mau berbagi dan membantu
- **Persistent** — tidak menyerah saat stuck

## Ekspektasi Anggota

### Untuk Siswa Aktif

**Komitmen minimal:**
- Hadir di pertemuan rutin (mingguan/dua mingguan)
- Kontribusi ke minimal 1 proyek per semester
- Dokumentasi pembelajaran (blog, GitHub README, dll)
- Membantu junior atau sesama anggota

**Yang akan kamu dapatkan:**
- Akses ke mentor dan alumni
- Kolaborasi di proyek nyata
- Persiapan kompetisi (OPSI, OSN, CTF, dll)
- Portofolio untuk universitas/industri
- Sertifikat kontribusi (jika konsisten)

### Untuk Alumni

**Peran:**
- Mentor untuk track tertentu
- Code review untuk proyek siswa
- Sharing session tentang kuliah/industri
- Koneksi ke peluang (internship, kompetisi, dll)

**Benefit:**
- Networking dengan generasi baru
- Kontribusi ke pendidikan
- Akses ke proyek-proyek menarik

## Kultur Foundation

### Do's ✅
- **Ask questions** — tidak ada pertanyaan bodoh
- **Share knowledge** — ajari yang kamu tahu
- **Fail fast, learn faster** — error adalah bagian dari proses
- **Document everything** — tulis apa yang kamu pelajari
- **Collaborate openly** — code review, pair programming
- **Give credit** — acknowledge kontribusi orang lain

### Don'ts ❌
- **Gatekeeping** — semua orang bisa belajar
- **Toxic competition** — kita berkembang bersama
- **Copy-paste tanpa paham** — belajar, jangan sekadar selesai
- **Ghosting** — komunikasi jika ada kendala
- **Elitism** — senior bukan lebih baik, hanya lebih dulu belajar
