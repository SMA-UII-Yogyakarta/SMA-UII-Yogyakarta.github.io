# Context Folder

Sebelum mengerjakan apapun, baca semua file di `context/`:

- `context/ACTIVE.md` — pekerjaan aktif dan backlog terkini
- `context/DECISIONS.md` — keputusan teknis yang sudah diambil
- `context/CONVENTIONS.md` — konvensi kode dan deploy
- `context/UI.md` — sistem desain dan mobile-first rules
- `context/INTEGRATIONS.md` — SLiMS API, auth flow, env vars

Folder ini adalah source of truth bersama antara developer dan AI.
Selalu update `context/ACTIVE.md` setelah menyelesaikan atau memulai pekerjaan baru.
